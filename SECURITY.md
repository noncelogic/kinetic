# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.x.x   | :white_check_mark: |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please email us at [security@noncelogic.com](mailto:security@noncelogic.com).

Include:

- Type of issue
- Steps to reproduce
- Impact of the issue
- Any potential fixes you've identified

We'll respond within 48 hours and work with you to understand and address the issue.
